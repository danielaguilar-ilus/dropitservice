# apps/web

Frontend React de Dropit.

## Vistas

- Login administrador.
- Dashboard de indicadores.
- Formulario cliente para solicitudes.
- Cotizaciones administrativas.
- Carga masiva Excel/CSV.
- Planificacion de rutas.
- Camiones y choferes.
- Tracking cliente.

## Variables

- `VITE_API_URL`
- `VITE_MAPBOX_TOKEN` para futuras vistas de mapa.
